1. What is a Stack?

Definition

A stack is a collection of objects that follows the Last-In, First-Out (LIFO) principle.

The last element added to the stack will be the first one to be removed.

Basic Operations

push(e): Add an element to the top of the stack
pop(): Remove and return the top element from the stack
top(): View the top element without removing it
size(): Get the number of elements in the stack
isEmpty(): Check if the stack is empty

2. Applications of Stacks

Common Use Cases

Web browser history ("Back" button):

Each visited page is pushed onto a stack. The "Back" button pops the stack to return to previous pages.

"Undo" mechanism in text editors:

Each action is pushed onto a stack. "Undo" pops the last action and reverses it.

Reversing an array:

Push all elements onto a stack, then pop them back into the array in reverse order.

Matching parentheses and HTML tags:

Stack helps validate proper nesting and matching of symbols.

3. Array-Based Stack

Implementation Details

Storage: Array data with fixed capacity N
Stack top: data[t] where t = size - 1
push(e): Increment t, assign e to data[t]
pop(): Retrieve data[t], set data[t] = null, decrement t

Advantages

Simple, efficient (O(1) for operations)

Disadvantages

Fixed capacity

Reference pages: 230, 231

Code Example


public class ArrayStack<E> {
    private E[] data;
    private int t = -1;
    
    public ArrayStack(int capacity) {
        data = (E[]) new Object[capacity];
    }
    
    public void push(E e) {
        if (size() == data.length)
            throw new IllegalStateException("Stack is full");
        data[++t] = e;
    }
    
    public E pop() {
        if (isEmpty()) return null;
        E answer = data[t];
        data[t] = null;
        t--;
        return answer;
    }
    
    public int size() { return t + 1; }
    public boolean isEmpty() { return t == -1; }
    public E top() { return isEmpty() ? null : data[t]; }
}

4. Singly Linked List Stack

Implementation Details

Storage: Singly linked list
Stack top: Front element of the list
Adapter Pattern: LinkedStack class contains SinglyLinkedList
push(e): list.addFirst(e)
pop(): list.removeFirst()

Advantages

Dynamic size, O(1) operations

Disadvantages

More memory for pointers

Reference page: 233

Code Example


public class LinkedStack<E> {
    private SinglyLinkedList<E> list = new SinglyLinkedList<>();
    
    public LinkedStack() {}
    
    public void push(E e) {
        list.addFirst(e);
    }
    
    public E pop() {
        return list.removeFirst();
    }
    
    public E top() {
        return list.first();
    }
    
    public int size() {
        return list.size();
    }
    
    public boolean isEmpty() {
        return list.isEmpty();
    }
}

5. Reversing an Array Using a Stack

Principle

The LIFO property of stacks naturally helps reverse the order of elements.

Steps

1. Push all array elements onto the stack
2. Pop elements from the stack and write them back into the array
Result: The array is reversed

6. Matching Parentheses & HTML Tags

Parentheses Matching

Check correctness of (), {}, []
Algorithm:
  • Encounter opening symbol: push
  • Encounter closing symbol: pop and match
  • End of string, empty stack means valid

function isValidParentheses(s) {
    let stack = [];
    let map = {')': '(', '}': '{', ']': '['};
    
    for (let char of s) {
        if (!map[char]) {
            stack.push(char);
        } else if (stack.pop() !== map[char]) {
            return false;
        }
    }
    return stack.length === 0;
}

HTML Tags Matching

Check pairs <name> and </name>
Similar to parentheses:
  • Push opening tag name
  • Pop and match with closing tag

function isValidHTML(html) {
    let stack = [];
    let tagRegex = /<(\/?)(\w+)[^>]*>/g;
    let match;
    
    while ((match = tagRegex.exec(html)) !== null) {
        if (match[1] === '') { // Opening tag
            stack.push(match[2]);
        } else { // Closing tag
            if (stack.pop() !== match[2]) {
                return false;
            }
        }
    }
    return stack.length === 0;
}
